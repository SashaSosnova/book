# Том Сойер — читай и зарабатывай

Браузерное приложение-компаньон к физической книге «Приключения Тома Сойера»
(перевод К. Чуковского, изд. «Махаон»).

Репозиторий: https://github.com/SashaSosnova/book

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173/

## Что умеет

- оглавление всех 35 глав + заключение
- факты, слова и пометки для чтения
- тесты с наградой 10 ₽ за верный ответ
- ключ из книги 🔑, чтобы нельзя было пройти тест без чтения
- не больше 2 ошибок, иначе тест не засчитан
- секретные подарки ★ в отдельных главах
- счёт, выплаты и режим родителя (PIN)
