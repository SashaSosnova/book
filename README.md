# Том Сойер — читай и зарабатывай

Веб-приложение для тестов по книге «Приключения Тома Сойера»
(перевод К. Чуковского, изд. «Махаон»).

Репозиторий: https://github.com/SashaSosnova/book

## Как пользоваться на планшете (рекомендуется)

1. Откройте на планшете Chrome (Android) или Safari (iPad):  
   **https://sashasosnova.github.io/book/**
2. Добавьте на домашний экран:
   - **Android Chrome:** меню ⋮ → «Установить приложение» / «На экран „Домой“»
   - **iPad Safari:** «Поделиться» → «На экран „Домой“»
3. Открывайте ярлык как обычное приложение.

Прогресс и монеты хранятся **в браузере этого планшета**. Один планшет — один счёт. Не очищайте данные сайта. Перед сменой браузера/сбросом сделайте бэкап в режиме родителя (`···`).

## Запуск на компьютере

```bash
npm install
npm run dev
```

Откройте http://localhost:5173/

## Что умеет

- оглавление всех 35 глав + заключение
- факты, слова и пометки для чтения
- тесты с наградой 10 ₽ за верный ответ
- ключ из книги 🔑
- не больше 2 ошибок, иначе тест не засчитан
- секретные подарки ★
- счёт, выплаты и режим родителя (PIN)
